    Theme upgrade

    How to upgrade the theme#

    Step 1#

    Backup your site (files, folders, database).

    Step 2#

    Upload and export theme-package.zip to your host (override all files and folders).

    Step 3#

    1. Login to your server

    2. Go to your Magento install dir

      cd <your Magento install dir>

    3. Update Magento components

      bin/magento setup:upgrade

      If prompted, also run bin/magento setup:di:compile

    4. Generate static content

      bin/magento setup:static-content:deploy -f

    Upgrade data v1.4 to v1.5#

    Follow the steps above, then:

    1. Upload and export update-block-data-1.5.zip to your host (override)

    2. Login to your backend and go to

      MAGEBIG > Martfury Theme > Choose Layout > IMPORT CONTENT
